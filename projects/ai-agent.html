Project Overview

The project consists of developing and implementing an Autonomous AI Agent System, orchestrated via LangGraph. The system aims to automate the intake of potential leads, the creation and sending of personalized prospecting messages, and the intelligent management of follow-ups. The agent is capable of analyzing the context of the lead, identifying absences, and suggesting or executing actions to resume contact, optimizing the sales flow.

Key Features

  • Sales Funnel Analysis
  • Contextual Personalization
  • Intelligent System
  • Supabase Management
  • Lead Dashboard

Technical Details

Unlike simple automations, this project uses a graph-based architecture. This allows the agent to have memory, make non-linear decisions, and customize communication based on unstructured data, making the approach scalable and adaptable to new sales scenarios.
