Project Overview

The project utilizes a Kaggle sales dataset that has been engineered to simulate a realistic CRM environment. Key features like credit risk, regional potential, number of employees, CNAE (industry code), and company size were created to serve as the foundation for the predictive model. The final model provides an automated, data-driven solution for client scoring, enabling a more efficient and targeted sales strategy.

Key Features

  • CRM Analysis
  • Customer Segmentation
  • Lead Scoring
  • Data Simulation & Feature Engineering
  • Customer Lifetime Value Prediction

Technical Details

Several popular machine learning models were trained and evaluated, including XGBoost, Random Forest, LightGBM, and Logistic Regression, to determine the best-performing algorithm for this classification task.
